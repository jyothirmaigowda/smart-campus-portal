# 🎓 Smart Unified Campus Portal

## 📌 Problem Statement
Campus services like attendance, assignments, and notices are scattered across multiple systems, causing inefficiency.

## 💡 Solution
A centralized platform integrating all academic services with real-time updates and smart insights.

## 🎯 Features
- Attendance tracking  
- Assignment submission  
- Notices & announcements  
- Student & faculty dashboard  
- Result management  

## 🚀 Unique Features
- Low attendance alerts  
- Assignment deadline reminders  
- Real-time notifications  
- Smart dashboard insights  

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js / Firebase  
- Database: MongoDB / Firebase
